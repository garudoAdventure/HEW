# ほっかいトレージャー
![ほっかいトレージャー](https://garudoadventure.github.io/portfolioWeb/assets/title3.png)

### 操作方法(キーボード)
| Input | Action |
| --- | --- |
| Up | 前進 |
| Right/Left | 回転 |
| Space | 大砲を打つ |

### マイクで遊ぶ準備
マイクを使ってプレイする場合は、外部マイクを接続し、設定で名前を「Microphone」に変更してください。
<img src="https://i.imgur.com/QWUzKM4.png" width="400"/>

マイクの設定画面を開いたままゲームを起動し、タイトル画面で息を吹くと、船が画面外に行き、ゲームが始まります。

### 操作方法(マイク)
| Input | Action |
| --- | --- |
| Right/Left | 回転 |
| 息を吹き | 前進 |
| 一瞬で強く吹き | 大砲を打つ |
